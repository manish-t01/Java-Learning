# Java Learning Journey

This repository contains all the code I have written during my journey of learning Java. 
It serves as a personal archive of my coursework and practice as a computer applications student, covering everything 
from fundamental syntax to core Object-Oriented Programming (OOP) principles and advanced topics.

## 📂 Repository Structure

Here is an overview of the topics and concepts covered in this repository, organized by directory:

### ☕ Java Basics & Control Flow
* **`JavaBasics/`**: Foundational concepts, syntax, and simple beginner programs.
* **`loopsInJava/`**: Practice with control flow using `for`, `while`, and `do-while` loops.
* **`arrayInJava/`**: Implementation of single and multidimensional arrays.
* **`methodsInJava/`**: Creating, invoking, and overloading functions.
* **`keywordsInJava/`**: Demonstrations of specific Java keywords (e.g., `static`, `final`, `this`, `super`).

### 📦 Object-Oriented Programming (OOP)
* **`classAndObjects/`**: Fundamentals of creating classes and instantiating objects.
* **`constroctorsInJava/`**: Implementation of default, parameterized, and copy constructors.
* **`getterAndSetter/`**: Using accessors and mutators to protect data.
* **`encapsulatin/`**: Bundling data and methods into a single protective unit.
* **`Inheritance/`**: Demonstrating code reusability through parent and child classes.
* **`abstractClass/`**: Implementing partial abstraction using abstract classes and methods.
* **`interfaceinjava/`**: Achieving abstraction and multiple inheritance using interfaces.

### ⚙️ Core Java Concepts
* **`exceptionhandling/`**: Managing runtime errors and preventing crashes using `try-catch-finally` blocks.
* **`multithereding/`**: Concurrent execution and process management using threads.

### 🧪 Standalone Scripts
* **`CpuStressTest.java`**: A standalone Java program for CPU testing and performance observation.
